## 使用说明

### 前端启动
在`frontend`目录下，执行如下命令，进行启动:

进入项目目录
```shell
cd frontend
```
安装Yarn，提示依赖的安装速度
```shell
npm install --global yarn
```
安装依赖
```shell
yarn install
```
启动服务
```shell
npm run dev
```

### 后端启动
- 修改`.yaml`文件里的`MySQL`数据库帐号密码
- 执行`sql`文件
- 运行`ServerApplication`启动类
